# Renal Cancer Survival Pipeline

## Overview

This project contains bioinformatics workflows developed during my MSc Bioinformatics research on renal cell carcinoma (RCC).

The study focused on identifying prognostic long non-coding RNAs (lncRNAs) through cross-cohort transcriptomic analysis and survival modeling.

## Research Objectives

- Identify downregulated lncRNAs in RCC
- Perform cross-cohort validation
- Evaluate prognostic significance using survival analysis
- Investigate potential biomarkers associated with disease progression

## Methods

- Transcriptomic data analysis
- Survival analysis
- Hazard ratio evaluation
- Log-rank testing
- Cross-cohort validation

## Tools and Technologies

- Python
- R
- Pandas
- Selenium
- OpenPyXL
- TCGA datasets
- ENCORI database
