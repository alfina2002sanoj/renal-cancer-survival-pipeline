# Renal Cancer Survival Pipeline

## Overview
This project contains bioinformatics workflows developed during my MSc Bioinformatics research on renal cell carcinoma (RCC).
The study focused on identifying prognostic long non-coding RNAs (lncRNAs) through cross-cohort transcriptomic analysis and survival modeling.

## Research Objectives
- Identify downregulated lncRNAs in RCC
- Perform cross-cohort validation
- Evaluate prognostic significance using survival analysis
- Investigate potential biomarkers associated with disease progression

## Methods
- TCGA transcriptomic data preprocessing
- Differential gene expression analysis
- Kaplan–Meier survival analysis
- Cox proportional hazards modeling (if applicable)
- Hazard ratio estimation
- Log-rank test for survival significance
- Cross-cohort validation (KIRC and KIRP)

## Key Findings
This study identified candidate long non-coding RNAs associated with survival outcomes in renal cell carcinoma (RCC) using TCGA KIRC and KIRP datasets.
These findings highlight potential prognostic biomarkers for further experimental validation.
